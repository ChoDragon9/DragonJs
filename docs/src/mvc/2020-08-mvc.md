## MVC
- MVC는 Model-View-Controller의 약자다.
- Model
  - 순수한 데이터 원형이다.
  - 데이터 변경을 전파한다.
- View
  - GUI이다.
  - Model 기반으로 렌더링한다.
  - 사용자 인터렉션 처리를 Controller에게 위임한다.
- Controller
  - Model과 View를 생성한다.
  - Model의 옵저버 역할을 한다.
    - Model에게 전달받은 데이터를 뷰에 전달한다.
  - View 사용자 인터렉션 처리를 한다.
